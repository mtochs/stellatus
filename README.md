# Parallax Pixel Stars

This project demonstrates a parallax effect using HAML and SASS.

## Setup

1. Install [HAML](http://haml.info) and [SASS](https://sass-lang.com) if you haven't already.
2. Compile the SASS to CSS:
   ```bash
   sass styles.sass styles.css
   ```
3. Convert HAML to HTML:
   ```bash
   haml index.haml index.html
   ```
4. Open `index.html` in your browser.
